# ROK AutoFarm - Product Features

## Overview

**ROK AutoFarm** is a desktop automation tool for Rise of Kingdoms. It automates daily tasks across multiple accounts, saving you hours of repetitive gameplay.

---

## Core Features

### Multi-Account Management
- Add and manage unlimited Rise of Kingdoms accounts
- Automatic login and account switching
- Secure credential storage
- Per-account task configuration

### Automated Collection

| Feature | Description |
|---------|-------------|
| **VIP Rewards** | Automatically claim daily VIP chest rewards |
| **Mail** | Collect all mail rewards with one click |
| **Alliance Rewards** | Claim alliance gifts and technology donations |
| **Daily Missions** | Complete and claim daily mission rewards |
| **Gold Chests** | Collect free gold chests on timer |
| **Legendary Chests** | Collect legendary chest rewards |

### City Development

| Feature | Description |
|---------|-------------|
| **Building Upgrades** | Automatically queue building upgrades |
| **Troop Training** | Keep barracks training continuously |
| **Technology Research** | Auto-queue technology research |

### Resource Farming
- Automated resource gathering on world map
- Support for all resource types: Food, Wood, Stone, Gold
- Smart troop dispatch and recall

---

## Task Queue System

### Smart Scheduling
- Automatic task rescheduling based on completion time
- Persistent queue - survives app restart
- Priority-based task execution

### Queue Controls
- **Run All** - Execute all tasks for all accounts
- **Pause/Resume** - Temporarily halt queue processing
- **Reset** - Clear all pending tasks

### Real-time Monitoring
- Live activity logs
- Current task status display
- Account progress tracking

---

## Screen Detection

### Template Matching
- OpenCV-powered screen recognition

- Automatic City/Map screen detection

### OCR Text Recognition
- Tesseract-based text extraction
